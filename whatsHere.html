What are the API's we used

We used 3 different API's in the making of this project the Nominatim API, the Leaflet API, and our own St. Paul Crime API. Each solves a different problem for us.

Nominatim

This API is an open-source geocoding API. What that means is that it is useful for findng locations on Earth by name and address. We used this to be able to pinpoint a crimes location and to be able to fit it into its neighborhood. This allows us to know how many crimes occured in each neighborhood

Leaflet

This API is an open-source JavaScript library that allows for mobile-friendly interactive maps. Leaflet let us find St. Paul and use it as out map for this project, which can be seen as the most pivotal part of the project. Leaflets' API was easy to use and allowed for a lot of customization that was otherwise not expected.

St. Paul Crime API

This API was created by us 3 using sqlite3 and data from the St. Paul government. This API allows us to find any crime we want that occured in St Paul for the past 10 years. Due to the nature of its creation we are able to customize it however we want, but with current capabilities are able to use many search parameters to narrow down to the exact types of crime we want.
