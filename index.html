Hello

This is a project made between Ian Schulke, Jesse Berndt-Alonso, and Zach Majorowicz. Using the Nominatim API, the Leaflet API, and our own St. Paul Crime API. The goal of this project was to map out crime across St. Paul, allowing us to be able to visualize crime statistics. This visualization not only helps us see where crimes are more prevalent, but also allow us to gain an understanding on why.
